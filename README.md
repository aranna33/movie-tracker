# 🎬 Movie Tracker

A simple full-stack demo app (Angular + FastAPI + PostgreSQL) for learning Docker, Docker Compose, and Kubernetes deployments.

---

## 🚀 Tech Stack

- **Frontend**: Angular
- **Backend**: FastAPI (Python)
- **Database**: PostgreSQL
- **Containerization**: Docker & Docker Compose
